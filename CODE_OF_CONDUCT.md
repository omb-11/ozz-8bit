# Code of Conduct

## Our Standard

Be respectful, specific, and technically honest. Disagreement is fine. Personal attacks are not.

## Expected Behavior

- Critique code and designs, not people.
- Document assumptions.
- Prefer reproducible bug reports and test cases.

## Unacceptable Behavior

- Harassment
- Doxxing
- Discriminatory language
- Deliberate sabotage of shared work

## Enforcement

Project maintainers may edit, remove, or reject contributions that violate these standards.
